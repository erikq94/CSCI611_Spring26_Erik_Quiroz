# Assignment 3 - Neural Style Transfer

## Description

Implementation of Neural Style Transfer based on Gatys et al using pretrained VGG19 network in PyTorch.

## Requirements

- Python 3
- PyTorch

## How to run

1.  Open Style_Transfer_Exercise_2.ipynb in Google Collab or Jupyter Notebook
2.  Upload your content image and style image.
3.  Run all cells from top to bottom.

## Files

- Style_Transfer_Exercise_2.ipynb - main notebook
- Images/ - content and style images
- Results/ - experiment output images
- Assignment 3 Style Transfer Report.pdf - report
